# 初信靈修記錄 App

一個專為初信耶穌的人設計的靈修記錄應用程式，包含8週、每週5天的靈修內容。

## 功能特色

- 📅 **8週靈修計劃**：完整的8週靈修旅程
- 📖 **每日經文**：每天都有幫助成長的經文
- 💭 **思考問題**：引導深入思考的問題
- 📝 **個人記錄**：四個記錄欄位
  - 經文主要內容
  - 個人感受
  - 實際應用
  - 禱告
- 💾 **自動儲存**：使用 localStorage 自動儲存所有記錄
- 📊 **進度追蹤**：顯示每週完成進度
- 🎨 **現代化 UI**：美觀易用的介面設計

## 安裝與執行

1. 安裝依賴：
```bash
npm install
```

2. 啟動開發伺服器：
```bash
npm run dev
```

3. 在瀏覽器中打開顯示的網址（通常是 http://localhost:5173）

## 建置生產版本

```bash
npm run build
```

建置後的檔案會在 `dist` 目錄中。

## 技術棧

- React 18
- Vite
- CSS3
- LocalStorage（資料持久化）

## 使用說明

1. 選擇週次：在主頁面選擇要進行的週次（1-8週）
2. 選擇日期：點擊該週的某一天（1-5天）
3. 閱讀經文：閱讀當天的經文和思考問題
4. 記錄靈修：在四個欄位中記錄你的靈修得著
5. 自動儲存：所有內容會自動儲存到瀏覽器的 localStorage

## 資料結構

所有靈修記錄都儲存在瀏覽器的 localStorage 中，即使關閉瀏覽器也不會遺失。資料格式為：

```json
{
  "1-1": {
    "week": 1,
    "day": 1,
    "mainContent": "...",
    "personalReflection": "...",
    "application": "...",
    "prayer": "...",
    "updatedAt": "2026-02-07T..."
  }
}
```

## 注意事項

- 資料儲存在瀏覽器的 localStorage 中
- 清除瀏覽器資料會導致記錄遺失
- 建議定期備份重要記錄
